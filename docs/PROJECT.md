For the "FuturaGame" repo please generate a basic java project framework for a 2d game. 
-- Project details --

This is a java project and needs a framework and classes set up automatically in order for us to complete the project.

-- Game details --

The game is going to be a 1v1 2d tank game in the same vein as "wii tanks" where 1 player controls a tank on a 2d plane and there is a computer controlled tank.

-- Tank Details --

The tanks fire projectiles and are able to move around in the 2d environment. The projectiles come out of the front of the tank

The tanks are able to rotate on their axis and move backwards and forwards based on where they are facing.

-- Environment details --

The environment is a "top down" look of the tanks and the landscape the tanks are maneuvering around
